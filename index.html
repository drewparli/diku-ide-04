Petur

I added colouring of individual toggled data points and rectangular selection. Using unique colors for data points helps with comparison of smaller sets of data points which have interesting correlations and promotes a richer analysis. The colouring is handled via a global pool of available colours. We decided to include as varied hues as possible while also avoiding using the other highlighting colours we have, like red and grey. To save the viewer from dealing with large sets of selected points with very minor differences in colour between them, we only add color to the first 5 points which are selected - the rest of coloured black.

The selection code is inspired by the solution found in this gist. Certain choices made in their code need to be implemented to adapt the solution, but the logic is the same. On the end event, when a selection has been made, the selection window's boundaries are checked and the colouring algorithm from before is employed.
